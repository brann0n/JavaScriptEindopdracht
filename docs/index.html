Uno Anywhere

Your regular UNO card game, now online!

The game works by using a laptop or tv as the game host, and then letting you connect with your mobile devices.

How can I play?

Go to the game lobby, from here you can either create a game or join an existing one.

How does it work?

If you create a game, you will be greeted with the host screen. This screen is what every player will be looking at during the game. On this screen you will see the current card, and the player expected to place a card.

If you join a game on your mobile device, you should start by holding the device in landscape mode as shown in the picture. Once the host starts the game, you will see your set of cards. Do not show them to anyone else!
